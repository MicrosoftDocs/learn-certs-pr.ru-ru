# Правила проведения экзаменов и часто задаваемые вопросы

Корпорация Microsoft обеспечивает наивысшее качество своих экзаменов и сертификаций, которыми пользуются партнеры и клиенты по всему миру. Для поддержки целей программы сертификации были разработаны правила проведения сертификационных экзаменов Microsoft.
